* [文件类用法](https://github.com/falowp-bot/falowp-bot-demo/tree/main/src/main/kotlin/com/blr19c/falowp/demo/plugins/file)

* [Web类用法](https://github.com/falowp-bot/falowp-bot-demo/tree/main/src/main/kotlin/com/blr19c/falowp/demo/plugins/web)
